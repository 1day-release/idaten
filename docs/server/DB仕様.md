# DB 仕様

| フィールド名 |              用途              |
| :----------: | :----------------------------: |
|   slide_id   | ランダムなスライド ID( 48 桁)  |
|    email     |    Google のメールアドレス     |
|   markdown   |  全てのマークダウンのテキスト  |
|  share_mode  | 0:非公開/1:公開閲覧/2:公開編集 |
|  create_at   |     スライドを作成した日時     |
|  update_at   |     スライドを更新した日時     |
